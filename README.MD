# Ferum Echo Bot

Бот для отправки сообщений всем пользователям анонимно.

## Установка

1. Склонируйте репозиторий: `git clone https://github.com/FerumDevelop/EchoBot`
2. Установите зависимости: `pip install -r req.txt`
3. Создать конфигурационный файл `.env` на основе `.env.example`
4. Запустить бота: `python bot.py`

## Структура

- `bot.py` - основной файл бота
- `admin.py` - функции администрации
- `user.py` - функции пользователей
- `database.py` - файл для работы с Базой Данных
- `keyboards.py` - файл с клавиатурами и интерфейсом бота
- `.env` - переменные
